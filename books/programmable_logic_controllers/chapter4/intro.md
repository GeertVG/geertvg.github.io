# Intro
---
As already mentioned, human-machine and machine-human interaction becomes possible by adding Human Machine Interfaces, or HMI for short. 
This interface mainly consists of:
-	Push buttons & switches
-	Potentiometers
-	Lamps
-	Light column
-	Horn of buzzer
-	BCD displays
-	Thumb wheel switches

However, these components can be expanded and/or replaced by using a programmable screen equipped with a processor, memory and the necessary network connections. The term ‘HMI display’ or ‘HMI screen’ is often used here to refer only to the screen.

![Siemens HMI](/images/Siemens_hmi.png "Siemens HMI display, ©2020 Siemens") 

Because an HMI screen is programmable, it offers the necessary flexibility; changes can be easily made to the interaction between human and machine and machine and human without having to make physical adjustments.

![Beckhoff HMI](/images/Beckhoff_hmi.png "Beckhoff HMI display, ©2020 Beckhoff") 

An HMI screen is characterised by:
-	The dimensions of the screen, with the size displayed in inches (diagonal distance)
-	Screen type: single-touch, multi-touch, presence of keys
-	The number of colours to be displayed (black and white, 16 colours, 256 colours, ...)
-	The type and number of network connections
-	The size of the internal memory
-	The possibility of adding external memory
-	Processor type
-	Resolution

![HMI display connections](/images/hmi_connections.png "HMI display connections, Basic panel, ©2020 Siemens") 

The combination of these characteristics, together with the manufacturer's sales model, may impose additional restrictions:
-	The number of PLC variables may be limited
-	The number of PLC connections may be limited
-	Some programmable functions may not be available because, for example, the screen is too small, the number of colours that can be displayed is too limited, ...
