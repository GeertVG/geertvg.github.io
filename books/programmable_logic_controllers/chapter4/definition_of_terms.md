# Definition of terms
---
| Term | Description |
| :---: | :---------- |
| HMI screen | An “Human Machine Interface” screen which is programmable and enables interaction between human-machine and machine-human |
| Single touch | Indicates that a touch screen can be operated with one finger |
| Multi touch | Indicates that a touch screen can be operated with multiple fingers at the same time, enabling functions such as zooming in and out |
| Static objects | Objects that do not change condition in RT and do not create a new action or condition when pressed |
| Dynamic objects| Objects that change condition in RT or that create a new condition and/or trigger an action by pressing the object |
